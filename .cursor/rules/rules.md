BEST PRACTICES:

- ALWAYS store types and interfaces in a suitable subfolder under the `types` directory.
- ALWAYS enable JWT tokens for Supabase functions in `config.toml`.
- ALWAYS follow best practices when implementing Supabase Auth, including secure handling of authentication and session data.
- ALWAYS use recommended patterns and best practices when using Zustand for state management.
- ALWAYS write idiomatic, maintainable, and performant React code.
- ALWAYS use modern TypeScript features and practices for type safety and clarity.
- ALWAYS organize routes and follow routing best practices when using React Router.
- ALWAYS follow React Hooks best practices (clean effect dependencies, custom hooks, etc.).
- ALWAYS use React Context thoughtfully for global state and configuration.
- ALWAYS use React Suspense for asynchronous data loading scenarios.
- ALWAYS implement error handling with React Error Boundaries where appropriate.
- ALWAYS use React Memo to optimize components for performance.
- ALWAYS use React Lazy for code splitting and improved loading performance.
- ALWAYS use React Fragments to avoid unnecessary DOM elements.
